## 项目介绍
基于java+jsp实现一个机票预定管理系统, 能够为用户提供预定航班查询与机票预定功能, 并实现一个管理员后台, 能够对用户航班订单进行管理

注意：一定要使用Mysql8版本，不然导入数据库会报错

## 项目技术
- 编程语言：Java
- 数据库：MySQL
- 前端技术：JSP、JavaScript、bootstrap、JQuery
- 后端技术：Servlet、JDBC

## 运行环境
- JDK版本：JDK1.8及以上
- 开发工具：IDEA、Ecplise、Myecplise都可以
- 数据库: MySQL8及以上
